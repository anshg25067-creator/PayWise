# Smart Debt Tracker(PayWise)
So this project is basically about that everyday problem which no one really talks properly like… you give money to your friends, or they give you, and then after few days everything become confusing.
who owes who? how much? when? and then awkward situation starts
so yeah this website is trying to solve that… but not in boring way.

## what is this actually?
this is not just “add expense and forget” type app. this is more like a smart system which understand what you are trying to say instead of forcing you to fill 10 fields.
like normal apps:
* enter name
* enter amount
* select type
* date
* blah blah
too much work honestly.

## so what is different here?
main thing is: you dont need to manually type everything properly
you can just say/type like normal human:
“gave rahul 500 for dinner”
and system will try to understand:
* who is person
* how much money
* did you gave or took
it feels little like chatting not filling form.

also there is some dashboard thing
so not just storing data randomly
you can see:
* who owes you most
* who you owe most
* trends like monthly
its like small analysis but useful.
